Detailed Description

driver definition for Statut LEDs Indicator (sli) module

This driver "SLI"(Statut LEDs Indicator) provide visual indication of harware statut. the driver supports 1 or 2 wired pins system and manage up to 3 LEDs indicator as detail below

2 pins wired (default)

STAT1 STAT0 Green Yellow Red Green Red
0 0 OFF OFF OFF OFF OFF
0 1 ON ON OFF OFF ON
1 0 OFF OFF ON ON OFF
1 1 ON OFF OFF ON ON

1 pin wired

STAT0 Green Red
0 OFF ON
1 ON OFF
